# highlight
Colorize text in your terminal :rainbow:

---

### Install via [pipx](https://pypa.github.io/pipx/):
```bash
  pipx install git+https://github.com/tspotts/highlight
```

### Usage:
```bash
  cat /tmp/rgb.log | highlight --red='foo' --green='bar' --blue='baz' 
```
